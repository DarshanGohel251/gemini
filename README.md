# 🚀 Gemini Space Race: Pro Engine

An AI-powered multiplayer space board game where strategy, fuel management, and real-time challenges decide the winner.

---

## 🎮 Features

- 🧑‍🚀 Multiplayer turn-based gameplay (2–4 players)
- ⛽ Fuel management system with dynamic consumption
- 🌀 Randomized board shuffling every few rounds
- 🤖 AI-based challenge system using Google Gemini API
- 🚀 Special tiles:
  - Boost (+Fuel)
  - Hazard (-Fuel)
  - Vortex (AI challenge)
  - Ultimate recharge (100% fuel)
  - Trap tile (penalty zone)
- 🎲 Animated dice system
- 🧠 Strategic decision-making gameplay

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom UI + Neon Theme)
- JavaScript (Vanilla)
- Google Gemini API
